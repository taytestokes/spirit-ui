import { Text } from "@spirit-ui/design-system/server";

import { CodePreview } from "../_components/code-preview/code-preview";

## Text

Display text using defined typography styles.

### Variants

The variant will determine how to the text is visually represented.

<CodePreview
  code={`
  <Text variant="extraLarge">Extra Large</Text>
  <Text variant="large">Large</Text>
  <Text variant="medium">Medium</Text>
  <Text variant="base">Base</Text>
  <Text variant="small">Small</Text>
  <Text variant="extraSmall">Extra Small</Text>
`}
>
  <Text variant="extraLarge">Extra Large</Text>
  <Text variant="large">Large</Text>
  <Text variant="medium">Medium</Text>
  <Text variant="base">Base</Text>
  <Text variant="small">Small</Text>
  <Text variant="extraSmall">Extra Small</Text>
</CodePreview>

### Color

The color of the text can be applied by passing the name of the color that is represented by our color tokens.

<CodePreview
  code={`
  <Text color="pink_700">Welcome to Spirit UI</Text>
`}
>
  <Text color="pink_700">Welcome to Spirit UI</Text>
</CodePreview>

### Weight

The font weight of the text can be applied by passing the name of the font weight through the weight prop.

<CodePreview
  code={`
  <Text weight="regular">Regular</Text>
  <Text weight="medium">Medium</Text>
  <Text weight="semibold">Semibold</Text>
  <Text weight="bold">Bold</Text>
  <Text weight="black">Black</Text>
`}
>
  <Text weight="regular">Regular</Text>
  <Text weight="medium">Medium</Text>
  <Text weight="semibold">Semibold</Text>
  <Text weight="bold">Bold</Text>
  <Text weight="black">Black</Text>
</CodePreview>

### Polymorphic

Override the underlying element that is rendered for the text to change it's semantic meaning. By default, the `p` element will be rendered.

<CodePreview
  code={`
<Text as="h2" variant="h1">
    Semantically rendered as {"<h2>"}, but visually rendered as {"<h1>"}
</Text>
`}
>

  <Text as="h2" variant="h1">Semantically rendered as {"<h2>"}, but visually rendered as {"<h1>"}</Text>
</CodePreview>
