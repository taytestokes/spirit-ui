# Text

Display text using defined typography styles.

## Variants

The variant will determine how to the text is visually represented.

<CodePreview
  code={`
  <Text variant="p">Paragraph</Text>
  <Text variant="p-sm">Paragraph Small</Text>
  <Text variant="label">Label</Text>
  <Text variant="label-sm">Label Small</Text>
  <Text variant="button">Button</Text>
  <Text variant="button-sm">Button Small</Text>
`}
>
  <Text variant="p">Paragraph</Text>
  <Text variant="p-sm">Paragraph Small</Text>
  <Text variant="label">Label</Text>
  <Text variant="label-sm">Label Small</Text>
  <Text variant="button">Button</Text>
  <Text variant="button-sm">Button Small</Text>
</CodePreview>

## Color

The color of the text can be applied by passing the name of the color that is represented by our color tokens.

<CodePreview
  code={`
  <Text color="blue_700">Blue</Text>
  <Text color="red_700">Red</Text>
  <Text color="yellow_700">Yellow</Text>
  <Text color="green_700">Green</Text>
  <Text color="teal_700">Teal</Text>
  <Text color="purple_700">Purple</Text>
  <Text color="pink_700">Pink</Text>
`}
>
  <Text color="blue_700">Blue</Text>
  <Text color="red_700">Red</Text>
  <Text color="yellow_700">Yellow</Text>
  <Text color="green_700">Green</Text>
  <Text color="teal_700">Teal</Text>
  <Text color="purple_700">Purple</Text>
  <Text color="pink_700">Pink</Text>
</CodePreview>

## Polymorphic

Override the underlying element that is rendered for the text to change it's semantic meaning. By default, the `p` element will be rendered.

<CodePreview
  code={`
<Text as="h2" variant="h1">
    Semantically rendered as {"<h2>"}, but visually rendered as {"<h1>"}
</Text>
`}
>

  <Text as="h2" variant="h1">Semantically rendered as {"<h2>"}, but visually rendered as {"<h1>"}</Text>
</CodePreview>
