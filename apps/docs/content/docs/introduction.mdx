# Introduction

Spirit UI is a design system and component library built with Typescript, React, and Vanilla Extract to support the development of Next.js applications by providing design guidelines and principles with reusable components that establish a unique and consistent feel for your applications.

{/* It takes inspiration, but not affiliated with, from the [Geist Design Sytem](https://vercel.com/geist/introduction) for design foundations and the [Mirror Design System](https://degen-xyz.vercel.app/) for architectural decisions. */}
